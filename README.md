# פרויקט LifeSnaps - עיבוד נתונים פיזיולוגיים ורגשיים (EMA & Fitbit)

ברוכות וברוכים הבאים לפרויקט! 
מאגר זה נועד לעזור לנו לנתח קובץ נתונים עצום (9.2 ג'יגה-בייט) של מדדים פיזיולוגיים (דופק, צעדים, HRV) ודיווחי רגש, מבלי לתקוע את המחשב. הקוד כאן יוריד את הקובץ ויפצל אותו לקבצים קטנים ומהירים (Parquet) של נבדקים בודדים.

## 📚 רקע על הנתונים (מקור המחקר)
הנתונים נלקחו מתוך מחקר ה-LifeSnaps. להבנה של סוגי המשתנים והמדדים, **חובה** לרפרף במאמר המקורי שפורסם ב-*Nature Scientific Data*:
👉 [LifeSnaps: A 4-month multi-modal dataset](https://www.nature.com/articles/s41597-022-01764-x)

---

## 🛠️ שלב א': התקנות חד-פעמיות (למי שאין עדיין סביבת עבודה)
אם מעולם לא תכנתם על המחשב הזה, עליכם להתקין 3 תוכנות חינמיות (זה לוקח 5 דקות):

1. **התקנת פייתון (Python):** 
   * היכנסו ל-[הורדת Python](https://www.python.org/downloads/) והורידו את הגרסה העדכנית.
   * ⚠️ **שלב קריטי בהתקנה:** במסך הראשון שקופץ, **חובה** לסמן ב-V את התיבה למטה שאומרת `Add Python to PATH`. בלי זה שום דבר לא יעבוד!
2. **התקנת Visual Studio Code (VSC):** 
   * זוהי התוכנה שבה אנחנו כותבים ומריצים קוד. היכנסו ל-[הורדת VS Code](https://code.visualstudio.com/) והתקינו כרגיל.
3. **התקנת Git:**
   * התוכנה שמאפשרת לנו למשוך את הקוד מהענן. היכנסו ל-[הורדת Git](https://git-scm.com/downloads) והתקינו (אפשר פשוט ללחוץ Next על כל החלונות, הגדרות ברירת המחדל מצוינות).

## 🚀 שלב ב': משיכת הפרויקט והפעלתו (Onboarding)

לאחר שסיימתם את ההתקנות, בצעו את השלבים הבאים כדי להביא את הפרויקט אליכם למחשב:

### 1. פתיחת הטרמינל (שורת הפקודה)
* פתחו את **VS Code**.
* בתפריט העליון, לחצו על **Terminal** ואז על **New Terminal**. ייפתח לכם חלון שחור בתחתית המסך.

### 2. העתקת הקוד למחשב שלכם
העתיקו את שתי השורות הבאות אל תוך הטרמינל (אפשר להדביק בעזרת קליק ימני בעכבר), ולחצו על `Enter` אחרי כל שורה. 
הפקודה הראשונה מורידה את הקוד שלנו, והשנייה מכניסה אתכם לתוך התיקייה:

```bash
git clone <https://github.com/MoriahTDaniel/lifesnaps_project.git>
cd lifesnaps_project
```

##⚠️ הערה חשובה לפני שמתחילים: התקנת ספריות בתוך VS Code
כדי שפנדס יוכל לשמור קבצים בפורמט Parquet וידע לקרוא BSON, חברות הצוות (ואת) תצטרכו להתקין 3 ספריות פשוטות בטרמינל של VS Code.
העתיקו את השורה הבאה לטרמינל ולחצו Enter:

```bash
pip install pandas pymongo pyarrow
```


### 3. הורדת קובץ המחקר הענק (9.2GB)
כעת נריץ את סקריפט ההתקנה. הוא יוריד אוטומטית את קובץ הנתונים הגולמי ממאגר המידע המדעי Zenodo למחשב שלכם.
העתיקו לטרמינל ולחצו Enter:
```bash
python setup_environment.py
```

הערה: יופיע לכם מד התקדמות באחוזים. ההורדה תיקח זמן (בהתאם לאינטרנט שלכם) – זה זמן מצוין להכין קפה!


### 4. חיתוך וסידור הנתונים
אחרי שההורדה הגיעה ל-100%, הריצו את סקריפט החיתוך.
הקוד ירוץ על קובץ ה-9GB, יחלץ את המדדים הפיזיולוגיים החשובים (דופק, צעדים ו-HRV) וישמור אותם מחולקים לפי נבדקים בתוך תיקייה חדשה בשם processed_data_parquet.
העתיקו לטרמינל ולחצו
```bash
python import_data.py
```

### 5. וידוא תקינות
אחרי שהקוד מסודרת בתקיות אנחנו רוצות לוודא שלכל חברות הצוות התקבל אותו מידע.
העתיקו לטרמינל והריצו:

```bash
python data_summary.py
```
זה קובץ שמסכם את הנתונים הקיימים.
